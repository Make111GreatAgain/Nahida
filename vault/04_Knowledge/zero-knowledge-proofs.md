# Zero-knowledge proofs

## 定义与范围

- 定义: Zero-knowledge proofs 研究证明者如何在不泄露见证或内部状态的前提下说服验证者某个陈述成立。当前 vault 覆盖 proof systems、polynomial commitments、recursion/folding、applications 与 zkML。

- Canonical terms: `zero-knowledge-proofs`
- Aliases/query keys: ZKP, 零知识证明
